Developer Documentation

This is a live working document outlining developer notes and documentation about actions, filters, and todos.
This plugin will be heavy commented so developers and follow along with every aspect of the development and flow of code. There will be no confiluded
codebase. Just Sweat and Simple.....

Plugin API Endpoints:

Authenticate: wp-json/ddm/v1/authenticate

Plugin Actions:

do_action( 'ddm_before_authentication_request', $request );
do_action( 'ddm_failed_authentication_request', $request );
do_action( 'ddm_after_authentication_request', $user, $access_token );

Plugin Filters:

$response = apply_filters( 'ddm_authentication_successfull_response', $response, $user );
